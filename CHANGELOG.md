# Changelog

## 1.7.0

- Prepared the project for public GitHub and browser-store release.
- Added polished English and Persian documentation, MIT License, contribution and security guides.
- Added store listing copy, reviewer notes, a publishing checklist and GitHub Pages privacy documents.
- Changed the public product name to “wrench — Minimal New Tab”.
- Disabled live Google suggestions by default in Chromium.
- Removed live remote autocomplete and its host permission from the Firefox build.
- Set Firefox minimum version to 140 and finalized a stable AMO extension ID.
- Added a tag-based GitHub release workflow.

## 1.6.1

- Replaced the multicolor Google suggestion badge with a monochrome black search icon.
- Removed secondary Google labels and right-side action text from live suggestions.
- Simplified the direct web-search result for a cleaner command palette.

## 1.6.0

- Added a required first-run profile dialog and a locally stored display name.
- Replaced the letter W in the header, popup and extension icons with a custom wrench mark.
- Added proper GitHub and Telegram SVG icons in the open-source settings footer.
- Changed the command-search shortcut to Ctrl/Command + Shift + K.
- Added live Google autocomplete suggestions with a 220 ms debounce, request cancellation and an in-memory cache.
- Added a Settings switch to disable live Google suggestions.
- Added the Google suggestion host permission and updated the privacy notice accordingly.

## 1.5.0

- Replaced the icon-only search action with an always-visible bottom command search.
- Added Ctrl/Cmd + K focus without scrolling or page rerendering.
- Added live suggestions for bookmarks, direct URLs, recent searches, and the selected web search engine.
- Added keyboard navigation with Arrow Up/Down, Enter, and Escape.
- Replaced the bottom toolbar glyphs with a consistent custom SVG icon set.
- Removed board rerendering while typing to keep large bookmark libraries responsive.

## 1.4.0

- Rebuilt the new-tab interface as a distinct, quiet and minimal wrench workspace.
- Added a lowercase wrench wordmark and a new extension icon.
- Replaced the duplicated legacy CSS layers with one coherent responsive stylesheet.
- Added a compact top workspace bar, calm three-column boards and a centered command dock.
- Kept board sections collapsible and context menus portaled outside the cards.
- Added an open-source footer in Settings with the project owner's GitHub and Telegram links.
- Improved Persian and English font presets with visible weight and spacing changes even when a preferred system font is unavailable.
- Preserved local-first storage, bilingual UI, bookmark-bar migration, focus/break modes and custom image/video backgrounds.

## 1.3.0

- Added compact glass bookmark boards and fixed floating context menus.
- Reduced full-page rendering during language and font changes.
