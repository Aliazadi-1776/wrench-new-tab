# wrench v1.7.0

A store-readiness and open-source release.

## Added

- GitHub-ready English and Persian READMEs
- MIT License, contributing and security guides
- Store listing, reviewer notes and publishing checklist
- GitHub Pages landing page and public privacy policy
- Automated release workflow for browser ZIP packages

## Privacy and store compliance

- Live Google suggestions are now off by default in Chromium.
- The Firefox build removes live remote autocomplete and the Google suggestion host permission.
- Firefox minimum version is set to 140 and declares no extension data collection.
- Store names, descriptions, homepage links and stable Firefox add-on ID were finalized.
