# Security Policy

## Supported version

Security fixes are applied to the latest published version of wrench.

## Reporting a vulnerability

Please do not open a public issue for an unpatched security vulnerability.

Contact the maintainer through Telegram at https://t.me/im_wrench with:

- A clear description of the issue
- Steps to reproduce
- Affected browser and version
- Potential impact
- A suggested fix, when available

Avoid including personal data, private bookmarks or credentials in reports.
