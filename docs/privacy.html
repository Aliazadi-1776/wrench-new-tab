Privacy Policy

Effective date: July 15, 2026

wrench is a local-first browser extension. The project owner does not operate an account system, analytics service, advertising platform or backend that receives user data.

Local data

Display name, pages, boards, links, notes, recent searches, settings, custom backgrounds and backups may be stored locally in the user's browser. Users can export or erase this data from wrench settings.

Permissions

  • storage saves workspace data locally.
  • activeTab is used only after the user invokes quick save.
  • contextMenus adds “Save to wrench”.
  • bookmarks is optional and requested only for user-initiated bookmark migration.

Web search

When the user submits a web search, wrench opens the selected provider with the query.

In Chromium, optional live Google suggestions are off by default. When explicitly enabled, typed text is sent directly to suggestqueries.google.com without browser credentials. The project owner does not receive or store queries or responses.

The Firefox store build does not include live remote autocomplete.

Data sharing

wrench does not sell user data. The project owner does not receive locally stored workspace data. A search provider receives a query only after an explicit search action, or when the Chromium user explicitly enables live suggestions.
